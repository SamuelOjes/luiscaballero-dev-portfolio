Motivation & Goals

I built up this project because I love wise quotes and wanted to share some wisdom on how to stay relaxed and motivated in these difficult times.


In order to practice my agile skills, I organized the whole project in a single sprint of two weeks. The sprint included the UX/UI design and the whole implementation of the app.


I created a Trello board with the assets, tasks, user journey, features and deadlines.


Tech Used

React hooks were use throughout the project for fetching the data and manage the states without the need for Redux.


To organize all the custom CSS I used SASS and the ITCSS architecture fundamentals. It helped me to organize the CSS files and deal with the CSS specifics.


Last but not least, I used PropTypes for the component props and I am building several integration tests with Jest and react-testing-library.

Problems & Thought Process

One of the biggest problem was actually finding the right REST API. There aren’t so much quote APIs out there that are free and with no need of an API_KEY. Lucky me, I found the Quote Garden REST API.


Other problems that came in my way were the built of a fetch reducer hook, how to deliver the data to the user and post it in Twitter and how to make the app accessible.

Lessons Learned

This project helped me to solidify my React and JavaScript ES6 knowledge, specially the use of hooks and how to manage the different states.


I got great practice on how to fetch data from an API, my first hands on testing and a much better understanding of accessibility.


Before this project, my knowledge and commitment to accessibility was suboptimal. Now I have a much better grasp about accessibility issues.
